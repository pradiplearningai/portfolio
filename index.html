HTMl

HTML, short for Hyper Text Markup Language, is the foundation of every webpage. It defines the structure and content of web pages, organizing text, images, and other elements using tags and elements that web browsers can interpret.

CSS

CSS, short for Cascading Style Sheet, is a style sheet language used to describe the presentation and styling of a document written in a markup language, most commonly HTML.

JS

JS, short for Java Script, is a high-level, interpreted programming language primarily used for creating dynamic and interactive web content.
